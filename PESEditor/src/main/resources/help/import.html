OF2

After a second option file (OF2) has been opened (using the Open OF2... option in the File menu), you can import the following data from it:

  • Individual Players - on the transfer page, double click the player you want to replace, click Import (OF2), and then double click the player you want to import
  • Individual Emblems - a new option will be available on the emblem page, and also when you click an existing Emblem
  • Individual Logos - a new option will be available when you click a slot on the Logo page
  • Individual Kits - double click a team on the team page, a list of the teams in OF2 will appear, double click the team who's kit you want to import. You will be prompted to choose a slot for each logo required by the kit

On the OF2 Import page:

  • Options / PES / Shop items / victory data
  • Stadium names
  • League names
  • Boots
  • Players / Squads / Formations
  • Club names
  • Kits / Emblems / Logos / Club stadiums, flags, colours

Note

Files from the US PC game should be named KONAMI-WIN32WEXUOPT before opening in the editor if to be used for import/export.
